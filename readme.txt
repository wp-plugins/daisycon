== Frequently Asked Questions ==


**Vraag 1 - Als ik op een link klik kom ik op ManyMoreStores.com in plaats van de pagina van de adverteerder.**

Dit kan twee dingen inhouden.

Je bent niet aangemeld bij de adverteerder. Log in bij Daisycon en meld de juiste website aan bij de adverteerder. Stuur een mail naar wordpress@daisycon.com mocht je problemen ondervinden bij het aanmelden
De tweede optie is dat je een foutief media ID hebt ingevuld. Het media ID bestaat alleen uit cijfers (b.v. 12345). [Lees hier waar je jouw media ID kan vinden.](http://www.daisycon.com/nl/faq/publishers/affiliate_marketing/waar_vind_ik_mijn_media_id/ "Media ID Daisycon")


**Vraag 2 - Er staat veel ruimte tussen mijn tekst en de vergelijker. Hoe krijg ik dat weg?**

Sommige content management systemen, zoals WordPress plaatsen automatisch enters die je niet kan weghalen. Je kan dit oplossen door de laatste regel

`<div id="daisyconSimonlyComperator" style="background-color:#FFFFFF !important;</div>`

te vervangen door

`<div id="daisyconSimonlyComperator" style="background-color:#FFFFFF !important; margin-top:-50px;"></div>`

De hoogte kan je zelf aanpassen door het getal bij "margin-top" te veranderen. Wij hebben hier gekozen voor -50 pixels. De vergelijker schuift dan 50 pixels omhoog.
